# Iris Foto Studio

Professional photography studio portfolio and online booking system. Specializing in product photography, portraits, events, and commercial shoots with a focus on high-quality deliverables and exceptional client service.

## Overview

Iris Foto Studio is a modern, responsive photography portfolio website with integrated online booking and client management. Showcase your best work, attract clients, and streamline the booking process all in one place.

## Features

### Current Features
-  **Portfolio Gallery** - Showcase photography work with high-resolution images
  - Portfolio sections: Product Photography, Portraits, Events, Commercial
  - Image filtering by category
  - Lightbox viewer for detailed image inspection
  
-  **Service Listings** - Display available services with pricing
  - Product Photography (e-commerce, catalog, lifestyle)
  - Portrait Sessions (individuals, families, headshots)
  - Event Coverage (birthdays, weddings, corporate events)
  - Commercial/Advertising shoots
  
-  **Online Booking System** - Allow clients to schedule sessions
  - Calendar-based date/time selection
  - Service selection with automatic pricing
  - Client information form
  - Email confirmation for both client and studio
  
-  **Responsive Design** - Works seamlessly on all devices
  - Mobile-first approach
  - Touch-optimized interface
  - Fast loading with image optimization
  - SEO-optimized for local search (Buzau Iris photography)
  
-  **Client Contact Form** - Direct messaging from potential clients
  - Inquiry tracking
  - Auto-response emails
  - Contact history

-  **Local SEO** - Optimized for local search

## Tech Stack

### Current
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** PHP 7.4+
- **Hosting:** Hostinger
- **Image Optimization:** Lazy loading, WebP format support

  - Google My Business integration
  - Structured data markup
  - Local schema implementation
